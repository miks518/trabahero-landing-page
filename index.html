Technology

Built With Modern Tools

Machine Learning

Scam Pattern Detection

Trabahero uses ML models trained on known job scam datasets to classify listings as high-risk, suspicious, or safe based on textual and structural features.

Browser Extension

WXT + React + TypeScript

Built as a modern Chrome extension using WXT, React 19, and TypeScript. Content scripts analyze DOM elements while the sidepanel provides the UI.

Visual Element Capture

DOM + Screenshot Pipeline

Every selected element is captured both structurally (tag, id, class, text) and visually (screenshot region) for comprehensive analysis.

Privacy

Privacy Policy

Effective Date: October 1, 2025

Trabahero is a Chrome extension that helps protect Filipino job seekers from employment scams. We designed Trabahero to analyze job listings primarily on your device, minimizing the data that leaves your browser.

What Trabahero Accesses

To provide job scam detection, Trabahero's content scripts access DOM content of job listing elements you explicitly select for analysis. We store user preferences (e.g., scan settings) locally via chrome.storage.

We do not read passwords, form inputs, private messages, or your browsing history beyond the elements you choose to scan.

Data Processing

Most analysis happens locally in your browser. When a listing requires advanced ML analysis, the selected element's text content is sent to our secure servers via HTTPS. No personal identifying information is included in these requests.

What We Don't Collect

We do not collect names, emails, passwords, financial information, location data, or browsing history. We do not use advertising networks or analytics services.

User Control

You can disable scanning at any time through the extension popup, adjust settings, or uninstall the extension to remove all local data.
