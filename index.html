A Universal Visual Job-Scam Detection System for Filipino Job Seekers

Trabahero is a Chrome extension that visually analyzes job listings in real-time — detecting scam patterns, matching your resume, and keeping you safe. All inside your browser sidebar.

How It Works

Protection in Three Steps

Trabahero lives in your browser and works alongside you as you browse job listings on any website.

touch_app
Step 1

Pick an Element

Click the Trabahero toolbar icon to open the popup, then launch the side panel. Use the element picker to visually select any job posting on the page — title, company, salary, or the entire listing card.

bolt
Step 2

AI-Powered Analysis

Trabahero's ML models analyze the element's content for scam patterns — unprofessional domains, vague salary ranges, AI-generated text, and suspicious contact information. A screenshot is captured for visual verification.

security
Step 3

Get Instant Results

View a detailed risk score with an animated gauge, a list of red flags with critical severity badges, and a full breakdown — all inside your browser sidebar. No page reloads needed.

Feature

Visual Element Picker

Select any DOM element on a page with a visual overlay. The content script communicates with the sidebar in real-time.

ads_click

Picker States

Idle • Selecting • Active • Selected

The picker button cycles through four states: idle ("Pick Element"), activating (pulsing "Selecting..."), active ("Cancel Selection" in red), and the selected element panel showing metadata and screenshot.

crop

Visual Overlay

Crosshair • Blue highlight

When active, a blue highlight overlay follows your mouse across elements. The cursor changes to crosshair. A toast notification confirms picker mode. Press Escape to cancel anytime.

screenshot_monitor

Screenshot Capture

Region crop • Data URL

Each selected element is captured as a cropped PNG screenshot via the browser's tab capture API. The screenshot is displayed alongside the element's tag name, ID, class, and text content.

Technology

Built With Modern Tools

Trabahero is built as a Manifest V3 Chrome extension with a React frontend and TypeScript throughout.

Framework

WXT + React 19 + TypeScript

Built with WXT (Next-gen Web Extension framework), React 19, and TypeScript. Tailwind CSS for styling with a custom dark "Sentinel Path" design system.

Permissions

activeTab • storage • tabs • sidePanel

Minimal Chrome permissions: activeTab for the current page, storage for preferences, tabs for tab communication, and sidePanel for the sidebar UI. Host permissions for <all_urls> and local dev server.

Architecture

4 Entrypoints • 19 Components

Four entrypoints (background, content script, popup, sidepanel) working together via Chrome's message passing API. Nineteen React components organized into shell, scan, match, and common groups.

Privacy

Privacy Policy

Effective Date: October 1, 2025

Trabahero is a Chrome extension that helps protect Filipino job seekers from employment scams. We designed Trabahero to analyze job listings primarily on your device, minimizing the data that leaves your browser.

What Trabahero Accesses

To provide job scam detection, Trabahero's content scripts access DOM content of job listing elements you explicitly select for analysis. We store user preferences (e.g., scan settings, auto-scan toggle) locally via chrome.storage. The background service worker opens the side panel when you click the toolbar icon.

We do not read passwords, form inputs, private messages, or your browsing history beyond the elements you choose to scan.

Data Processing

Most analysis happens locally in your browser. When a listing requires advanced ML analysis, the selected element's text content is sent to our secure servers via HTTPS. No personal identifying information is included in these requests.

What We Don't Collect

We do not collect names, emails, passwords, financial information, location data, or browsing history. We do not use advertising networks or analytics services.

User Control

You can disable auto-scanning at any time through the toggle in the scan view, adjust settings, or uninstall the extension to remove all local data.

For questions or concerns, please reach out through the Support Center above.
