# Trabahero Landing Page

Premium landing page for **Trabahero** - a Chrome extension that protects Filipino job seekers from online employment scams by scanning job postings in real-time.

Built with Vite + React + TypeScript + Tailwind CSS.

## Setup

```bash
npm install
npm run dev       # local dev server
npm run build     # production build to dist/
```

## Tech Stack

- **Vite** — build tool
- **React 19** + TypeScript
- **Tailwind CSS 3** — PostCSS plugin
- **Google Fonts** — Inter, Hanken Grotesk, Material Symbols
